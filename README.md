# main-site-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能涉及不同主题或用途，彼此之间没有强关联。仓库的主要目的是提供一个集中存放和展示 HTML 内容的平台，方便访问与维护。

## 目录说明

仓库根目录下直接存放 HTML 文件，每个文件对应一个独立的页面。目录结构保持扁平，不设子文件夹。

```
/
├── index.html         # 首页/导航页（可选）
├── page-a.html        # 示例页面 A
├── page-b.html        # 示例页面 B
└── ...
```

`index.html` 可作为总览页，列出所有已归档的页面链接，便于导航。其他 HTML 文件直接以文件名标识内容。

## 页面归档说明

- 每个 HTML 文件均为自包含的完整页面，无需依赖外部资源（除非在文件内显式引用）。
- 文件命名建议使用简洁的英文或拼音，避免特殊字符。
- 页面内容应遵守相关法律法规，不包含违法或不良信息。
- 归档时请注意文件大小，保持页面轻量，便于快速加载。

## 维护说明

- 新增页面：直接向仓库提交新的 HTML 文件，并更新 `index.html`（如有）。
- 更新页面：修改对应 HTML 文件内容后提交。
- 删除页面：移除文件并更新索引。
- 欢迎通过 Issues 或 Pull Requests 提出改进建议或报告问题。
